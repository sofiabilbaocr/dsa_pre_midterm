REPRODUTOR DE MÚSICA
Este proyecto implementa una Lista Doblemente Enlazada para gestionar una playlist de 50 canciones reales como parte del simulacro de parcial.


Como ejecutar el proyecto:
1.Clona mi repo:
Asegurate de tener los archivos ll_double_linked.py y playlist_demo.py en la misma carpeta.

2:Abre una terminal y ejecuta el comando:
python playlist_demo.py

-Controles del Reproductor:
n: Avanzar a la siguiente cancion.

p: Regresar a la cancion anterior.

q: Salir del programa.
